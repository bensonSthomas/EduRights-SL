EduRights SL API Documentation

Open Educational Resource API for Sierra Leone

Overview

The EduRights SL API provides public access to approved educational resources while promoting legal and ethical educational content sharing.

Data is provided in JSON format and can be integrated into websites, mobile applications, digital libraries, and research platforms.

Base URL

http://127.0.0.1:5000

Available Endpoints

Method Endpoint Description
GET /api/resources Returns all approved resources
GET /api/resources?category=Books Filter by category
GET /api/resources?type=PDF Filter by resource type
GET /api/resources?search=LAW Search resources by title
GET /api/resources/1 Return one specific resource

Example Response

{
    "project": "EduRights SL",
    "data_standard": "JSON",
    "total_resources": 2,
    "resources": [
        {
            "id": 1,
            "title": "IT LAW LECTURES",
            "category": "Books",
            "resource_type": "PDF",
            "license_type": "Creative Commons"
        }
    ]
}
